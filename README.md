# [Biblical Chronology](https://cyblight.github.io/Biblical_Chronology/)

Современный одностраничный сайт с интерактивной схемой библейской хронологии.

## Что реализовано

- Интерактивная лента событий с выбором карточек
- Фильтры по эпохам и текстовый поиск
- Синхронизированная SVG-схема маршрутов/регионов
- Адаптивная верстка для desktop и mobile

## Запуск

1. Перейдите по ссылке нажав на этот текст: [Biblical Chronology](https://cyblight.github.io/Biblical_Chronology/)

## Файлы

- `index.html` — структура страницы
- `styles.css` — визуальный стиль и адаптив
- `script.js` — данные событий и интерактивная логика

## Примечание

Визуальный подход вдохновлен форматом библейских карт и схем, используемых на jw.org, без копирования исходных материалов.
